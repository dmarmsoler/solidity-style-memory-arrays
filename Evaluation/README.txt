This folder contains tests and results for Certora, solc-Verify and solCMC provers (Table 3 in paper). 

	1- There are three sub-folders: Certora, solc-verify and solCMC.
	2- Each sub-folder consists of two more subfolders: tests and results, and a dedicated README file. 
	3- tests folders contain solidity (.sol) test smart contacts used for each prover. 
	4- results folders consists of files (.txt) containing results corresponding to each test in the tests folder.
	5- Each folders README provides guide for reproducing the results for the tests in corresponding provers.
